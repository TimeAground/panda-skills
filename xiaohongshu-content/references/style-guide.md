# Xiaohongshu Style Guide

## Title Rules

1. Under 20 chars ideal, max 25
2. Must include number OR emoji OR question
3. No English technical terms
4. Test: would your mom click on this?

## Title Formulas

- Pain point + solution: "Never miss a page update again, this tool watches for you"
- Number + benefit: "25MB AI that does voice-over, free"
- Story + curiosity: "After missing visa appointment, he built this"
- Comparison + surprise: "Free tool beats the 99/month subscription"

## Body Rules

1. First 2 lines decide everything (hook)
2. One idea per paragraph
3. Use emoji bullets not dashes
4. Break long sentences into 2 lines
5. Speak like a person not a manual
6. End every post with a question

## Words to Avoid

- GitHub (say "open source community" or just skip)
- API / CLI / SDK (explain in plain words)
- Parameters (say "settings")
- Deploy (say "install" or "use")
- Repository (say "project")

## Words that Work

- Free, no cost
- One-click, simple, easy
- Hidden gem, little-known
- Tested personally, hands-on
- Life-changing, game-changer
- AI magic

## Hashtag Strategy

Always include 3 tiers:
- Tier 1 (big): #AI tool #free tool #efficiency tool (100k+ posts)
- Tier 2 (medium): category specific (10k-100k posts)
- Tier 3 (small): product name, niche terms (under 10k posts)
